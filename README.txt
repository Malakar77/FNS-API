Параметры запроса:
Параметр	Тип данных и пример	Описание
q	string
q=Гордиенко Василий	Поисковая строка. Если указать q=any, то будут выведены все организации, соответствующие параметру filter
page	integer
page=2	Страница поиска (поиск выдает только первые 100 найденных значений, если их больше используйте page для отображения следующей порции найденных значений)

Необязательное поле
filter	string
filter=active	Фильтрация вывода. Возможно указать следующие значения (разделяя символом +):
active - выводит только действующие организации
onlyul - выводит только юридические лица (без ИП)
onlyip - выводит только ИП (без юридических лиц)
okved - выводит организации с заданными основным ОКВЭД, разделенными символом | (пример okved69.10|69.20 - выведет организации с основным ОКВЭД 69.10 или 69.20)
okvedgroup - выводит организации с заданной группой ОКВЭД (пример okvedgroup69 - выведет организации с основным ОКВЭД 69,69.1,69.10,69.20,69.20.2,69.20.3 и т.д.)
region - выводит организации, зарегистрированные в заданных регионах (пример region77|78 - выведет организации Москвы и Санкт-Петербурга)
vyruchka - выводит организации с выручкой в заданном интервале (пример vyruchka>5000<20000 - выведет организации с выручкой от 5 до 20 млн. руб.)
sotrudnikov - выводит организации с количеством сотрудников в заданном интервале (пример sotrudnikov>5<20 - выведет организации с сотрудниками от 5 до 20 человек)
datereg - выводит организации с датой регистации в заданном интервале (пример datereg>01.01.2019<01.01.2018 - выведет организации, зарегистрированные в 2018 году)
reestrmsp - выводит организации, входящие в реестр МСП на последнюю доступную дату
reestrmsp1, reestrmsp2 или reestrmsp3 отбирает Микро, Малые или Средние предприятия соответственно
withphone - выводит организации, у которых указан номер телефона в контактах
withemail - выводит организации, у которых указан email в контактах




//возвращаемый 
Возвращаемый документ (HTTP response) представляет собой структурированный документ JSON, содержащий список найденных организаций.
Поля возвращаемого документа:
Имя	Тип	Описание
items	array	Массив результатов поиска
ЮЛ, ИП или НР	object	Тип найденной компании (юридическое лицо, индивидуальный предприниматель или представительство иностранной компании)
ИНН	string	ИНН найденной компании
ОГРН	string	ОГРН найденной компании
НаимСокрЮЛ	string	Наименование ЮЛ полное (только для ЮЛ)
НаимПолнЮЛ	string	Наименование ЮЛ краткое (только для ЮЛ)
ФИОПолн	string	ФИО ИП (только ИП)
ДатаРег	string	Дата регистрации ЮЛ (ИП) в формате YYYY-MM-DD
Статус	string	Статус ЮЛ (ИП): «Действующее», «Ликвидировано», «В состоянии реорганизации» и др.
ДатаПрекр	string	Дата прекращения деятельности ЮЛ (ИП) (если деятельность прекращена) в формате YYYY-MM-DD
АдресПолн	string	Адрес организации (полный)
ОснВидДеят	string	Основной вид деятельности
ГдеНайдено	string	Где найдена поисковая строка:

    ОГРН, ИНН,
    ОГРНИП, ИННИП,
    ИНН учредителя, ИНН бывшего учредителя,
    ИНН директора, ИНН бывшего директора,
    ИНН руководителя, ИНН бывшего руководителя (если руководитель не первый),
    Наименование ЮЛ,
    ФИО учредителя, ФИО бывшего учредителя,
    ФИО директора, ФИО бывшего директора,
    ФИО ИП,
    Адрес,
    Участие в УК с указанием доли,
    Фильтр.

nextpage	object	true, если в запросе есть следующия страница
filter	object	Строка фильтра, если запрос с параметром filter
filter_any_count	integer	Общее количество найденных записей при фильтрации (выводится если указан параметр filter и req=any)
Count	integer	Количество найденных записей
